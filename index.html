HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The elements in the body elements are known as child elements
  • The footer elements holds the footer content

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border creates a border around an element
  • A box-shadow property adds box-shadow effects around an elements frame
  • Colors in CSS can be assigned in many ways, including by semantic references or by its hexadecimal code

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A adds starter code to the branch we ar currently working on
  • git commit -m "commit message" commits the changes with a message that describes what the commit contains
  • code . opens VS Code on the computer

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
